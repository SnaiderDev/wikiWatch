# Wiki<span style="color:#1EC6D9">Watch</span>

WikiWatch es una aplicación web que permite buscar y explorar información de películas y series. Está diseñada para ofrecer una experiencia visual moderna y ágil, usando React para el frontend y Tailwind CSS para el diseño.

## 🚀 Tecnologías usadas
- 🧩 **React**: Librería principal para construir la interfaz de usuario.
- ⚡ **Vite**: Herramienta de compilación rápida y moderna.
- 🎨 **Tailwind CSS**: Framework de utilidades para estilos responsivos y personalizados.
- 🔗 **React Router DOM**: Navegación entre páginas de películas, series e información detallada.
- 🪄 **React Icons**: Iconos integrados para mejorar la experiencia visual.

## ✨ Características principales
- Búsqueda de contenido en tiempo real.
- Visualización de detalles de películas y series.
- Interfaz responsiva y moderna.
- Navegación fluida entre secciones.

## 🧩 Estructura del proyecto
- `src/App.jsx`: Componente principal de la aplicación.
- `src/main.jsx`: Punto de entrada de Vite.
- `src/components/`: Componentes reutilizables para la UI.
- `src/hooks/`: Hooks personalizados para obtener datos y lógica de búsqueda.
- `src/style/style.css`: Estilos globales y configuraciones de Tailwind.
- `src/config/`: Configuración de API y textos.

## ⚙️ Cómo ejecutar el proyecto
1. Instala dependencias:
   ```bash
   npm install
   ```
2. Inicia la aplicación en modo desarrollo:
   ```bash
   npm run dev
   ```
3. Abre el navegador en la dirección que indique Vite.
